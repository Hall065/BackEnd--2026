# 📦 Sistema Administrativo - Conceito Visual (Warehouse UI)

## 🎯 Objetivo
Criar um sistema administrativo com identidade visual inspirada em:
- Galpões logísticos
- Linhas de produção
- Embalagens de papelão
- Movimentação de estoque

Foco:
> Experiência visual moderna + temática industrial (não genérico igual ERP comum)

---

## 🎨 Identidade Visual

### 🟤 Cores principais
- Bege papelão: `#D9B382`
- Marrom kraft: `#A67C52`
- Cinza metálico: `#5F6B73`
- Cinza azulado: `#3E4A52`
- Fundo claro: `#F5F1E8`

### 🎯 Cores de destaque
- Verde status: `#4CAF50` (estoque ok)
- Amarelo alerta: `#FFC107` (baixo estoque)
- Vermelho erro: `#F44336` (problema)

---

## 🧱 Conceito de UI

### 🏗️ Estilo geral
- Cards = caixas de papelão
- Bordas levemente irregulares (simulando corte)
- Sombras suaves (como caixas empilhadas)
- Textura leve de papel

### 🧩 Elementos visuais
- Ícones estilo stencil / industrial
- Linhas tracejadas (como marcação de embalagem)
- Etiquetas tipo envio (labels)

---

## 🧭 Layout Base

### 📌 Sidebar (Menu lateral)
- Cor: Cinza metálico
- Ícones simples (outline)
- Hover = efeito "pressionado" (como botão físico)

Itens:
- Dashboard
- Produtos
- Estoque
- Clientes
- Fornecedores
- Pedidos

---

### 📊 Dashboard

#### Componentes:
- Cards com métricas:
  - Total de produtos
  - Estoque baixo
  - Pedidos ativos
  - Clientes

💡 Visual:
- Cada card = caixa com etiqueta colada
- Pequeno ícone como "carimbo"

---

## 📦 Página de Produtos

### Estrutura:
- Grid de cards (caixas)
- Cada produto:
  - Nome
  - SKU
  - Quantidade
  - Status

💡 Interação:
- Hover → caixa "levanta"
- Clique → abre detalhes como se abrisse a caixa

---

## 🚚 Página de Pedidos

### Ideia:
Simular fluxo logístico

- Status:
  - Recebido
  - Separando
  - Enviado

💡 UI:
- Timeline horizontal estilo esteira

---

## ⚙️ Animações (ESSENCIAL)

### 🏭 Linha de produção (Hero / Dashboard)

- Esteira animada (loop)
- Caixas andando da esquerda → direita

### 🤖 Garras mecânicas
- Movimento vertical pegando caixas
- Pode ser usado em loading

### 📦 Interações
- Hover em card:
  - leve escala + sombra
- Drag & Drop:
  - caixa "flutua"

---

## 🌌 Parallax

### Camadas:
1. Fundo (galpão desfocado)
2. Esteiras
3. Caixas
4. UI (cards)

💡 Movimento leve no scroll

---

## 🧠 Microinterações

- Botões:
  - efeito "pressionado"
- Loading:
  - caixa sendo montada
- Sucesso:
  - caixa sendo selada ✔️

## 🧪 Diferencial (IDEIAS AVANÇADAS)

- Sons sutis:
  - clique = caixa
  - hover = metal leve

- Modo "dark industrial":
  - fundo escuro + aço + luz fria
